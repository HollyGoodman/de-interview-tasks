Data engineering task

Scenario- you are working for a B2B online retailer. You have all of your order data in a csv file, however all of your customer data is sent over in a json format. In order to provide your data analysts the correct data, you'll need to construct an ETL pipeline in a programming language of your choice to cleanse and join your data into an appropriate and easy to use dataset for your data analysts to use. 


Steps: 

- Create the cleaned and joined dataset ready for analysis 
- Import the dataset into your BI tool of choice to answer the following questions 	
	- What sales rep generated the most revenue? 
	- How much (in GBP) did O'Reilly-Braun buy over the period? 
	- How much (in GBP) was spent on the following skew: EXOB69C38TH
